# 智能语音识别系统使用说明

## 🎯 系统概述

智能语音识别系统是一个根据网络状态自动选择最佳识别方式的语音交互系统。

### 🌐 在线模式（有网络时）
- **主要识别方式**: 百度语音API
- **优势**: 中文识别准确率高、稳定
- **特点**: 云端处理，识别效果好

### 💻 离线模式（无网络时）
- **备用识别方式**: 本地语音识别
- **优势**: 无需网络、隐私性好
- **特点**: 本地处理，保护隐私

## 🔄 智能切换机制

### 网络监控
- **自动检测**: 每30秒检测一次网络状态
- **实时切换**: 网络状态变化时自动切换识别模式
- **无缝体验**: 用户无需手动干预

### 识别策略
```
有网络 → 使用百度API识别
  ↓
识别失败 → 自动切换到本地识别
  ↓
无网络 → 直接使用本地识别
```

## 📖 详细使用指南

### 语音识别使用

#### 基本操作
1. **启动系统**: 运行智能语音识别系统
2. **自动检测**: 系统自动检测网络状态
3. **开始录音**: 点击"🎤 开始语音识别"按钮
4. **智能识别**: 系统根据网络状态选择最佳识别方式
5. **查看结果**: 识别结果自动显示在界面上

#### 识别模式说明
- **🌐 在线模式**: 显示"使用百度API识别"
- **💻 离线模式**: 显示"使用本地识别"

#### 使用技巧
- **清晰发音**: 说话要清晰，语速适中
- **环境安静**: 尽量在安静的环境中使用
- **网络稳定**: 确保网络连接稳定以获得最佳识别效果
- **备用方案**: 即使网络异常，本地识别仍可工作

### 语音合成使用

#### 自动播放
- 当启用语音对话时，AI回复会自动播放语音
- 语音内容会自动去除HTML标记，只播放纯文本

#### 手动播放
- 在演示界面可以手动输入文本进行语音播放
- 支持调整语音参数（速度、音量）

## 🔧 系统配置

### 网络配置
```python
# 网络检测设置
NETWORK_CHECK_INTERVAL = 30  # 网络检测间隔（秒）
NETWORK_TIMEOUT = 3         # 网络连接超时（秒）
```

### 语音参数配置
```python
# 语音识别参数
ENERGY_THRESHOLD = 4000     # 能量阈值
DYNAMIC_ENERGY_THRESHOLD = True  # 动态能量阈值
PAUSE_THRESHOLD = 0.8       # 暂停阈值

# TTS参数
TTS_RATE = 180             # 语音速度
TTS_VOLUME = 0.8           # 音量大小
```

### 百度API配置
```python
# 百度语音API配置
BAIDU_API_KEY = "your_api_key"
BAIDU_SECRET_KEY = "your_secret_key"
```

## 🛠️ 故障排除

### 常见问题

#### 1. 网络连接问题
**症状**: 系统显示离线模式
**解决方案**:
- 检查网络连接
- 确认防火墙设置
- 尝试访问百度服务
- 检查代理设置

#### 2. 百度API识别失败
**症状**: 在线识别无法工作
**解决方案**:
- 检查API密钥配置
- 确认API调用限制
- 检查网络连接
- 系统会自动切换到本地识别

#### 3. 本地识别失败
**症状**: 离线识别无法工作
**解决方案**:
- 检查本地语音识别库安装
- 确认麦克风权限
- 检查音频设备设置
- 尝试重新安装依赖

#### 4. 麦克风问题
**症状**: 无法录音或录音异常
**解决方案**:
- 检查麦克风硬件连接
- 确认麦克风权限设置
- 关闭其他占用麦克风的程序
- 测试麦克风设备
