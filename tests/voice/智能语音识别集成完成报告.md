## 🚀 使用方法

### 在主程序中使用语音功能

1. **启动主程序**:
   ```bash
   python visualization_test2.py
   ```

2. **启用语音功能**:
   - 在界面中勾选"启用语音对话"复选框
   - 语音输入按钮将变为可用状态

3. **使用语音识别**:
   - 点击"🎤 语音输入"按钮开始录音
   - 系统会根据网络状态自动选择识别方式
   - 识别结果会自动填入对话输入框

4. **查看识别状态**:
   - 语音状态标签会显示当前识别模式
   - 🌐 在线模式 - 百度API
   - 💻 离线模式 - 本地识别
