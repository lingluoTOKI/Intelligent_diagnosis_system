# 开发板AI语音对话功能说明

## 🤖 功能概述

开发板的"🤖 AI语音对话"按钮现已完全与PC端的个性化输入框和DeepSeek AI系统集成，实现了完整的端到端语音对话体验。

## 🔄 工作流程

### 1. 语音录制阶段
```
用户点击"🤖 AI语音对话"按钮
↓
开发板开始5秒录音
↓
显示"正在录音中，请说话..."状态
```

### 2. 语音处理阶段
```
录音完成
↓
语音识别（目前为模拟识别）
↓
显示"正在处理语音，请稍候..."状态
```

### 3. PC端AI处理阶段
```
发送语音文本到PC端
↓
PC端接收命令（通过CommandListener）
↓
调用DeepSeek API进行AI对话处理
↓
生成医疗相关的智能回复
```

### 4. 回复播放阶段
```
PC端发送AI回复到开发板
↓
根据音频设备设置选择播放方式
↓
开发板TTS播放 / PC端播放 / 双端播放
↓
图形界面显示AI回复内容
```

## 📡 网络通信协议

### 语音命令格式
```json
{
    "type": "voice_command",
    "command": "voice_text",
    "params": {
        "text": "医生，我眼睛经常干涩，是什么原因造成的？",
        "source": "board_voice_recognition",
        "purpose": "ai_chat",
        "timestamp": "2025-01-14T01:30:00.000Z"
    },
    "timestamp": "2025-01-14T01:30:00.000Z",
    "source_addr": ["0.0.0.0", 5003]
}
```

### AI回复格式
```json
{
    "type": "ai_response",
    "text": "眼睛干涩可能由多种原因引起...",
    "timestamp": "2025-01-14T01:30:05.000Z",
    "audio_device": "board"
}
```

### 播放设备选择
- **PC端播放**：AI回复仅在PC端播放
- **开发板播放**：AI回复仅在开发板播放
- **双端播放**：AI回复同时在两端播放

- 支持中文语音合成

## 🚀 使用方法

### 图形界面操作
1. 点击"🤖 AI语音对话"按钮
2. 看到"正在录音中，请说话..."提示
3. 对着麦克风说出医疗相关问题
4. 等待"正在处理语音，请稍候..."
5. 听取AI回复或查看屏幕显示

### 控制台操作
1. 输入命令 `r`
2. 按提示进行语音录制
3. 等待AI处理和回复

### 摄像头预览中
1. 按键盘 `r` 键
2. 进行语音对话
3. 继续预览或退出
