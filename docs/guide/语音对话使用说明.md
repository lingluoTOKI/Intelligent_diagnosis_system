# 语音对话使用说明

## 功能概述
AI眼科疾病智诊系统现已支持语音交互功能，用户可以通过语音与AI进行自然对话，获得智能医疗建议。

## 主要特性
- 🎤 **语音识别**: 支持中文和英文语音识别，优先使用Google API（需要网络），本地Vosk作为备选
- 🔊 **语音播放**: AI回复自动转为语音播放，支持中文TTS
- 💬 **智能对话**: 语音识别结果自动发送给AI，实现连续对话
- 📊 **状态提示**: 实时显示语音识别状态和进度

## 使用步骤

### 1. 启用语音功能
- 在右侧AI建议区域找到"启用语音对话"复选框
- 勾选后，语音输入按钮将被激活

### 2. 设置API密钥（推荐）
- 输入DeepSeek API密钥以获得更专业的医疗建议
- 没有API密钥也可以使用默认建议

### 3. 开始语音对话
- 点击"🎤 语音输入"按钮
- 等待"正在录音，请说话..."提示
- 清晰地说出您的问题或症状描述
- 系统会自动识别并发送给AI
- AI回复会自动播放语音

## 语音识别支持
- **在线识别**: Google API（中文/英文）- 推荐，识别准确率高
- **离线识别**: Vosk模型（可选）- 无需网络，需要下载模型

## 状态说明
- 🎤 **正在录音**: 请清晰说话
- 🔄 **处理中**: 正在识别语音
- ✅ **识别成功**: 已填入对话框
- 📢 **播放中**: AI回复正在播放

## 常见问题

### Q: 语音识别失败怎么办？
A: 
1. 确保网络连接正常（优先使用在线识别）
2. 检查麦克风权限和设备连接
3. 在安静环境中清晰说话
4. 如果持续失败，可以手动输入文字

### Q: 如何提高识别准确率？
A:
1. 在安静环境中使用
2. 距离麦克风适当距离（20-30cm）
3. 说话清晰，语速适中
4. 避免方言，尽量使用普通话

### Q: 可以使用离线识别吗？
A: 可以，但需要安装Vosk并下载中文模型：
1. 安装依赖：`pip install vosk`
2. 下载模型：从 [Vosk官网](https://alphacephei.com/vosk/models) 下载 `vosk-model-small-cn-0.22`
3. 解压到程序目录或设置环境变量 `VOSK_MODEL_PATH`

### Q: TTS播放有问题怎么办？
A:
1. 检查系统音量设置
2. 确认扬声器或耳机连接正常
3. 重启程序重新初始化TTS引擎

## 技术规格
- **语音格式**: WAV, 16kHz采样率
- **录音时长**: 最长8秒
- **超时设置**: 10秒无声音自动停止
- **TTS限制**: 单次播放最长300字符

## 隐私说明
- 在线语音识别会将音频发送到Google服务器
- 本地Vosk识别完全在本地处理
- 不会保存任何语音录音文件
- AI对话内容仅用于生成回复，不会存储

## 建议和反馈
如果您在使用过程中遇到问题或有改进建议，请通过以下方式联系：
- 检查控制台输出的[DEBUG]信息
- 确认麦克风和网络状态
- 尝试重启应用程序
