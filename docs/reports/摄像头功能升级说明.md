# 摄像头功能升级说明

## 📸 升级概述

基于JupyterLab环境的需求，我们对开发板摄像头功能进行了全面升级，从原来的简单捕获模式升级为多线程实时视频流处理系统。

## 🔄 主要改进

### 1. 线程化视频流处理
**原版问题**: 
- 单线程阻塞式图像捕获
- 无法实现实时预览
- 帧率不稳定

**升级方案**:
```python
class CameraThread(threading.Thread):
    """独立的摄像头线程"""
    def __init__(self, callback=None):
        super(CameraThread, self).__init__()
        self.working = True
        self.running = False
        self.frame_callback = callback
        self.last_frame = None
        self.cap = None
```

**优势**:
- ✅ 独立线程处理视频流
- ✅ 非阻塞实时预览
- ✅ 稳定的帧率控制
- ✅ 自动错误恢复

### 2. 智能摄像头管理
**原版问题**:
- 摄像头断开后无法自动重连
- 资源释放不完整
- 状态监控缺失

**升级方案**:
```python
def init_camera(self):
    """智能初始化摄像头"""
    try:
        self.cap = cv2.VideoCapture(0)
        if not self.cap.isOpened():
            print("[错误] 无法打开摄像头")
            return False
        
        # 优化设置
        self.cap.set(cv2.CAP_PROP_FRAME_WIDTH, CAMERA_WIDTH)
        self.cap.set(cv2.CAP_PROP_FRAME_HEIGHT, CAMERA_HEIGHT)
        self.cap.set(cv2.CAP_PROP_FPS, CAMERA_FPS)
        self.cap.set(cv2.CAP_PROP_BUFFERSIZE, 1)  # 减少延迟
        
        return True
    except Exception as e:
        print(f"[错误] 摄像头初始化失败: {e}")
        return False
```

**优势**:
- ✅ 自动重连机制
- ✅ 完整的资源管理
- ✅ 实时状态监控
- ✅ 优化的参数设置

```python
def enhance_image(self, frame):
    """多步骤图像增强"""
    try:
        # 1. 双边滤波去噪
        denoised = cv2.bilateralFilter(frame, 9, 75, 75)
        
        # 2. 锐化处理
        kernel = np.array([[-1,-1,-1], [-1,9,-1], [-1,-1,-1]])
        sharpened = cv2.filter2D(denoised, -1, kernel)
        
        # 3. 自适应直方图均衡化
        lab = cv2.cvtColor(sharpened, cv2.COLOR_BGR2LAB)
        l, a, b = cv2.split(lab)
        clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
        l = clahe.apply(l)
        enhanced = cv2.merge([l, a, b])
        enhanced = cv2.cvtColor(enhanced, cv2.COLOR_LAB2BGR)
        
        return enhanced
    except Exception as e:
        print(f"[警告] 图像增强失败，使用原图: {e}")
        return frame
```

**优势**:
- ✅ 专业级图像去噪
- ✅ 细节锐化增强
- ✅ 色彩对比度优化
- ✅ 医疗级图像质量

## 🔧 技术细节

### 线程同步机制
```python
def run(self):
    """摄像头线程主循环"""
    self.running = True
    print("[摄像头] 视频流线程启动")
    
    while self.working:
        try:
            if not self.cap or not self.cap.isOpened():
                print("[警告] 摄像头未打开，尝试重新初始化...")
                if not self.init_camera():
                    time.sleep(1)
                    continue
            
            ret, frame = self.cap.read()
            if not ret:
                print("[警告] 图像获取失败")
                time.sleep(0.1)
                continue
            
            self.last_frame = frame
            
            if self.frame_callback:
                self.frame_callback(frame)
            
            time.sleep(1.0 / CAMERA_FPS)  # 帧率控制
            
        except Exception as e:
            print(f"[错误] 摄像头线程错误: {e}")
            break
    
    self.running = False
    print("[摄像头] 视频流线程停止")
```

### 资源管理优化
```python
def stop(self):
    """优雅停止摄像头"""
    print("[摄像头] 正在停止视频流...")
    self.working = False
    
    # 等待线程安全退出
    while self.running:
        time.sleep(0.1)
    
    # 释放资源
    if self.cap:
        self.cap.release()
        print("[摄像头] VideoCapture已释放")
```

### 状态监控系统
```python
def get_camera_status(self):
    """详细状态信息"""
    if self.camera_thread:
        return {
            "initialized": True,
            "running": self.camera_thread.running,
            "camera_opened": self.camera_thread.is_camera_opened(),
            "preview_active": self.is_previewing,
            "resolution": f"{CAMERA_WIDTH}x{CAMERA_HEIGHT}",
            "fps": CAMERA_FPS,
            "last_frame_time": time.time()
        }
    else:
        return {"initialized": False}
```
