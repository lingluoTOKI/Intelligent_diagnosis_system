# 开发板-PC端整合医疗诊断系统 - 完整使用指南

## 🎯 系统概述

本系统成功整合了开发板和PC端的医疗诊断功能，实现了**纯图片传输模式**的智能诊断系统，具有以下特点：

### ✨ 核心特性
- **📸 按需图片传输**：只在拍照诊断时传输，无持续视频流
- **🔊 双端音频播放**：AI建议可选择在PC端/开发板/双端播放
- **🎤 智能语音对话**：开发板语音输入 → PC端AI处理 → 智能回复
- **💾 自动图片保存**：拍照自动保存到PC端指定目录
- **🖥️ 简化界面**：开发板界面简洁明了，专注核心功能

## 🚀 快速启动

## 📋 功能详解

### 1. 📸 拍照诊断功能

#### 操作流程：
1. **开发板端**：点击"📸 拍照诊断"按钮
2. **自动拍照**：系统自动捕获摄像头图像
3. **图像传输**：图片分片传输到PC端
4. **AI诊断**：PC端AI模型分析图像
5. **结果返回**：诊断结果和建议返回开发板
6. **语音播报**：根据设置播放AI建议

#### 技术特点：
- **高质量压缩**：JPEG质量85%，保证诊断精度
- **可靠传输**：UDP分片传输，自动重组
- **自动保存**：图片自动保存到PC端指定目录
- **实时反馈**：传输状态实时显示

### 2. 🎤 语音对话功能

#### 操作流程：
1. **开发板端**：点击"🎤 语音对话"按钮
2. **语音录制**：录音5秒（可配置）
3. **语音识别**：转换为文本（目前为模拟）
4. **AI处理**：PC端DeepSeek大模型处理
5. **智能回复**：生成医疗相关建议
6. **语音播放**：根据设置选择播放设备

#### 支持的问题类型：
- 眼部疾病咨询
- 症状分析询问
- 预防措施了解
- 治疗建议获取

### 3. 🔊 音频设备管理

#### 设备选择选项：
- **PC端播放**：仅在PC端扬声器播放
- **开发板播放**：仅在开发板扬声器播放
- **双端播放**：同时在两端播放（演示模式）

#### 设置方法：
1. 在PC端点击"开发板交互管理"
2. 选择"音频播放设备设置"
3. 选择合适的播放模式
4. 点击"测试音频播放"验证

## 🔧 配置说明

### 网络配置
```python
# 默认端口配置
CAMERA_PORT = 5002      # 摄像头数据传输
DIAGNOSIS_PORT = 5003   # 诊断结果接收
COMMAND_PORT = 5004     # 命令控制
VOICE_SEND_PORT = 5005  # 语音数据发送
VOICE_RECEIVE_PORT = 5006 # 语音合成接收
```

### 图片保存配置

```python
# PC端保存路径
PC_SAVE_PATH = r"/ultralytics-main/datasets/test"

# 图片质量设置
JPEG_QUALITY = 85  # JPEG压缩质量
MAX_PACKET_SIZE = 1400  # UDP包最大大小
```

### 音频配置
```python
# 音频参数
CHANNELS = 1            # 单声道
RATE = 16000           # 采样率16kHz
RECORDING_DURATION = 5  # 录音时长5秒
```

## 🚨 故障排除

### 常见问题解决

#### 1. 图片保存失败
- **症状**：显示"图像已保存"但PC端无文件
- **解决**：检查保存目录权限，确保路径存在
- **验证**：运行 `test_image_transfer.py` 检查目录状态

#### 2. 网络连接问题
- **症状**：开发板显示"PC连接断开"
- **解决**：检查防火墙设置，确保端口5002-5006开放
- **验证**：在开发板点击"🔗 连接测试"

#### 3. 音频播放问题
- **症状**：AI建议无声音播放
- **解决**：检查音频设备设置，测试TTS功能
- **验证**：在PC端"开发板交互管理"中测试音频

#### 4. 摄像头无法使用
- **症状**：显示"摄像头未就绪"
- **解决**：检查摄像头连接，确保无其他程序占用
- **验证**：重启系统，优先启动本程序

## 💡 使用技巧

### 最佳实践
1. **拍照技巧**：
   - 保持摄像头稳定
   - 确保充足光线
   - 拍摄对象居中

2. **语音对话**：
   - 说话清晰，语速适中
   - 使用医疗相关术语
   - 避免环境噪音

3. **设备选择**：
   - 演示时选择"双端播放"
   - 私人使用选择"PC端播放"
   - 开发板独立使用选择"开发板播放"

### 性能优化
- **图片质量**：可根据需要调整JPEG_QUALITY（70-95）
- **录音时长**：可根据需要调整RECORDING_DURATION（3-10秒）
- **网络优化**：确保开发板和PC在同一局域网
