
### 2. 百度API错误处理

**问题**: 
```
[ERROR] 百度API请求失败: 401 - {"error_description":"unknown client id","error":"invalid_client"}
```

**原因**: 百度语音API的密钥配置无效或缺失。

**解决方案**: 
- 创建了详细的API配置说明文档
- 提供了多种配置方法
- 实现了智能降级机制

## 🔧 解决方案详情

### API配置方法

#### 方法一：直接配置
在代码中直接设置API密钥：
```python
self.baidu_api = BaiduSpeechAPI("您的API Key", "您的Secret Key")
```

#### 方法二：配置文件
创建 `api_config.json` 文件：
```json
{
    "baidu_api_key": "您的百度API Key",
    "baidu_secret_key": "您的百度Secret Key"
}
```
